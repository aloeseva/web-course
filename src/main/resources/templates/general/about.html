Проект Шумбасова Всеволода, студента 3 курса 2 группы.

1.

Образовательное веб-приложение. Задача: разработка клиент-серверного веб-приложения для ведения образовательной деятельности (размещения учебных материалов и проверка усвоеных знаний в тестовой форме) с дальнейшим размещением на веб-хостинге sprinthost.

2.

Для решения поставленной задачи будет проведено изучение и анализ аналогичных платформ и методов онлайн обучения. На основе полученных на прошлом этапе информации и собственных идей будет спроектирована модель предметной области. На следующем этапе, основываясь на модели составленной на прошлом этапе, начнеться реализация приложения, используя навыки и знания полученные за 2 года обучения на ФКН и самостоятельному изучению веб технологий. Для реализации приложения будут использованны языки: Python, JS, HTML, CSS, а так же сопутствующие технологии и фреймворки: Flask, Jinja, SQLAlchemy и другие при необходимости. Для хранения данных будет использоваться реляционная система управления базами данных - MySQL. По завершении реализации веб-приложения будут проведены необходимые приготовления и развертывание его на веб-хостинге sprinthost. Далее будет составлен отчет в котором будет описанна задачи и результат проделанной работы.

3.

Языки: Python, JS, HTML, CSS.

СУБД: MySQL.

Инструмент для визуального проектирования баз данных: MySQL Workbench.

Cпопутсввующие технологии и фреймворки: Flask, Jinja, Bootstrap, SQLAlchemy.

Интегрированная среда разработки: PyCharm.

Cистема управления пакетами: Pip.

4.1.

Функционал приложения: Регистрация пользователей, разделение пользователей по ролям, создание новых курсов, добавление учебных материалов, создание тестов, создание вопросов, прохождение тестов, сохранение результатов тестов, подсчет итоговой оценки, просмотри успеваемости.

4.2.

Интерфейс: Навигационная панель сверху, в которой имеються ссылки: на все курсы размещенные на сайте, на мои курсы (курсы на которые записался пользователь), на страницу с информациоей о проекте(эту страницу). Для учителей и администраторов предусмотрена возможность создавать новые курсы с интерфейсом для этого. В созданных курса предусмотрены кнопки для размещения материалов и тестов. В тестах можно создавать вопросы выбирая из тип в блоке select. Для пользователей (студентов) создан интерфейс в котором они могут скачивать выложенные материалы и проходить размещенные тесты.
